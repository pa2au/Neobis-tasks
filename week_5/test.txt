Программа подключена к базе данных week4 и реализовано 5 методов

Методы:
1) getConnection() - устанавливает соединение с базой данных с помощью DriverManager через пользователя и его пароль
2) showTable(String table) - выводит на консоль все значения таблицы. Для этого в параметре надо указать имя таблицы.
			     При успешном выводе на консоле отображается нужная таблица.
3) deleteClient(int id) - удаляет все значения клиента в таблице clients. Требуется ввести id нужного клиента
			  При успешном выводе на консоле отображается id + " was deleted" 
4) addCake (...) - добавляет новый элемент в таблицу kulikovsky. В параметре указаны значения, которые нужны заполнить.
		   При успешном выводе на консоле отображается "new cake was added"
5) updateCake(String name, int id) - обновляет элемент, изменяя его имя. Требуется ввести то имя, которое мы ходим увидеть, и его id
				     При успешном выводе на консоле отображается id + " was updated"

В программе реализовано:
deleteClient(6) - удаляет клиента с id=6 из таблицы clients
showTable("clients") - выводит таблицу clients на консоль
addCake(7779,"strawberry","cake","circle",800,"2021-05-10","2021-05-15") - добавляет элемент в таблицу kulikovsky
showTable("kulikovsky") - выводит таблицу kulikovsky на консоль
updateCake("Princess",7779) - меняет содержимое записи с id=7779 из таблицы kulikosky
